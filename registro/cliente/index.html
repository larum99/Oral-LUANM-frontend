Reglas de agendamiento

Validaciones activas para proteger la agenda compartida.

Domingos y festivos bloqueados No se permiten citas en domingos ni festivos oficiales de Colombia.
Horario compartido Si un especialista ya tiene una cita en la misma fecha y hora, el sistema no deja duplicarla.
Sincronizacion local lista para backend La cita queda guardada en el almacenamiento local y preparada para integrarse luego con el backend.
